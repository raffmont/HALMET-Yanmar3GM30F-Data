# NMEA 2000 mapping

The project uses the NMEA2000 library plus the ESP32 TWAI backend. HALMET CAN pins are defined in `include/board_pins.h` and exported to the NMEA2000 backend before including `NMEA2000_esp32.h`.

## Transmitted PGNs

| PGN | Name | Period | Data sent |
| ---: | --- | ---: | --- |
| 127488 | Engine Parameters, Rapid Update | 100 ms | Engine instance, engine speed in RPM. Boost pressure and tilt/trim are sent as N2K not-available values. |
| 127489 | Engine Parameters, Dynamic | 500 ms | Engine instance, oil pressure, coolant temperature, alternator voltage, and engine discrete-status bits. Unknown fields are sent as N2K not-available values. |
| 127505 | Fluid Level | 2500 ms | Fuel tank instance, fuel type, level in percent, configured capacity in liters. |

## D1-D4 panel warning mapping

| HALMET input | Default type-B panel meaning | NMEA 2000 field |
| --- | --- | --- |
| D1 | Low oil pressure lamp/alarm | PGN 127489 Engine Status 1: `LowOilPressure` |
| D2 | High coolant temperature lamp/alarm | PGN 127489 Engine Status 1: `OverTemperature` |
| D3 | Alternator charge lamp | PGN 127489 Engine Status 1: `ChargeIndicator` |
| D4 | Key-on / buzzer / spare | Not transmitted by default; used locally for engine state. |

When any D1-D3 warning is active, PGN 127489 also sets Engine Status 1 `CheckEngine`.

## Instance configuration

Configured in `include/yanmar_config.h`:

- `N2K_ENGINE_INSTANCE`
- `N2K_FUEL_TANK_INSTANCE`
- `N2K_FUEL_TANK_CAPACITY_L`
- `N2K_DEFAULT_NODE_ADDRESS`

## Unit conversion

Signal K engine speed is published in Hz. PGN 127488 is sent in RPM because the NMEA2000 library helper expects RPM for engine speed.
